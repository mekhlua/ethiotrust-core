Our Services

Face Recognition API

Compare two faces, detect liveness, prevent spoofing. 98.5% accuracy on diverse datasets.

POST /v1/verify/face

ID Verification API

Extract name, FAN number, DOB, and phone from Ethiopian IDs. OCR with 97.8% accuracy.

POST /v1/verify/id

Digital Card API

Issue virtual cards, process payments, manage balances. Tokenized for security.

POST /v1/cards/issue

Authentication API

Federated SSO, OAuth2, JWT tokens. One login for all your apps.

POST /v1/auth/token

Tokenization API

PCI-compliant data tokenization. Replace sensitive data with secure tokens.

POST /v1/security/tokenize

Quick Start

Get started with EthioTrust in 3 simple steps

1. Get your API Key
Use key: eth_et_e0d628e448e2db36049522a0f302b964
# Verify a face
curl -X POST http://localhost:8001/api/v1/verify/face \
  -H "X-API-Key: eth_et_e0d628e448e2db36049522a0f302b964" \
  -F "selfie=@selfie.jpg" \
  -F "id_photo=@id.jpg"
# Issue a digital card
curl -X POST http://localhost:8002/api/v1/cards/issue \
  -H "X-API-Key: eth_et_e0d628e448e2db36049522a0f302b964" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"user_123","identity_verified":true}'
# Process a transaction
curl -X POST http://localhost:8002/api/v1/cards/authorize \
  -H "X-API-Key: eth_et_e0d628e448e2db36049522a0f302b964" \
  -H "Content-Type: application/json" \
  -d '{"card_token":"CARD-xxx","amount":1500,"merchant":"Bank"}'
